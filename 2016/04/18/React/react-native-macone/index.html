    问题处理

    1.SDK location not found

    >  Failed to notify ProjectEvaluationListener.afterEvaluate(), but primary configuration failure takes precedence.
>  java.lang.RuntimeException: SDK location not found. Define location with sdk.dir in the local.properties file or with an ANDROID_HOME environment variable.

    这是因为ANDROID_HOME环境变量没有设置正确

    > * 请回看上面的内容

    2.failed to find target with hash string ‘android-23’ in: /usr/local/Cellar/android-sdk
    在android-sdk目录中不存在android-23(可能会是其他版本)
    办法:在Android SDK manager 中安装这个版本的android sdk build－tools
