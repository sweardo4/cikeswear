冬日

发表于 2014-11-07   |   分类于 记录

童年的某年某天开始冬日的太阳是那麽的温柔，照在脸上像是依偎在一个毛绒玩具身上一样，软软的暖暖的。这样的感觉也只能在冬日才能感受得到。
每年的四季更迭让我们的感受不同，你喜欢了冬季，我喜欢了春季。因此你我 也都有了盼望。
早起，寒冷天气，看见外面的景色有些惊喜，本是以为昨夜下了雪，然而那不是雪，只是霜。
“不知庭霰今朝落，疑是林花昨夜开。”

南王地-浮萍立冬水

发表于 2014-03-20   |   分类于 记录

这是一个美丽的地方，正如你跟着以诺出去支教的地方一样，这也是一个难忘的地方，但和那灯红酒绿的大城市相比多的是那一份幽静淡雅之气，春夏秋冬人们围绕的是这山，围绕的是这里的梯田。
这个地方人家是极少的，或是陶潜所描写的桃花源也不得知，我现在也只认识一人而已，她是个女子，虽生自这样的偏远的地方，但是从小家人也是诗书教育，随就有了大家闺秀之气，举止得礼之为。她的另一方面我是极喜欢的，之所以喜欢皆因我对妻子的理解：对生活里的知识了解很多，关心人是那样的恰当，这是在我之上的，那烧的一手好饭更是不在话下，且又是娇羞可爱，温文尔雅，一看到就会给人一种熟悉，和其他女子是极不相同的。有很多人也是和我的感觉是一样的，所以不知是哪个诗人为了讨她欢喜，特为她写了一首诗云：

诗云云 :

  • 初春阳光，
  • 夏季微风，
  • 中秋果实，
  • 寒冬炭火，
  • 愚人儿只识是全部，
  • 然未知不及她一半。

南王地-仙石化成人

发表于 2014-02-03   |   分类于 记录

今日有一仙石在这地方幻化成人，变得和我认识那个女子一样年纪，也巧不知怎么没过多久他们却成了好盆友，整日在山中一起玩耍，并没有因为出生而变的存在隔阂，就像出生在同一娘胎一样关系友好。只见的那仙石面容如白色海棠花一样，依稀有些淡红夹在其中，秀丽而端庄。
我是时常去这地的， 就这样仙石在这里生活的久了我们也就认识了，在这里我是对她有很多说不出的事情来，一次我是见到她哭了，也是我在这里第一次见到女孩哭，且哭的是那样的伤心，我不知道如何劝慰她，也不好意思去问到底发生了什么事情，我怕提起来她哭的更厉害了。
我们也是时常在一起玩耍，记得在一个集市上，她曾送我一个五彩的手环，那个手环我一直收藏者，有时我也带出来让别人看看，但是每一次她都不会出现。
我曾问过她喜欢读什么书？她说《红》，最喜欢的作家是张大小姐那个为爱而疯狂的传奇女子，巧的是我也是喜欢的，也不知她是什么时候读的，因此我觉察到我们的喜好大都是相同。
有些时候感觉我在故意远离这些人，也在故意远离这个地方，总感觉分离的日子进了，先远离一点别离之痛也就少一点吧！

有诗云云 :

  • 相聚的终将相聚，别离的终将别离。
  • 唱的是分离后的相聚，饮得是天落下的眼泪。
